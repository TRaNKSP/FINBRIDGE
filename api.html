Authentication

All requests require your API key as a Bearer token in the Authorization header. Keys are generated from the Partner Portal.
# Example header
Authorization: Bearer fb2030_sk_AbCdEfGhIjKlMnOpQrStUvWxYz0123456789ABCD
POST/auth/validate
{
  "api_key": "fb2030_sk_...",
  "partner_id": "PTNR-ABC123-DEF456"
}

// Response
{
  "valid": true,
  "partner_id": "PTNR-ABC123-DEF456",
  "org_name": "Acme Community Bank",
  "environment": "sandbox",
  "strategies_enabled": ["gig-micro-credit", "harvest-advance", "bridge-fund"],
  "quota_remaining": 48291,
  "expires_at": "2027-04-13T00:00:00Z"
}

  "currency": "USD",

Webhooks

Subscribe to real-time events for loan status changes, distributions, AI score updates, and overdraft alerts.
POST/webhooks/subscribe
// Request
{
  "url": "https://your-server.com/finbridge-webhook",
  "events": [
    "loan.approved",
    "loan.repayment_made",
    "bridge.distribution_issued",
    "ai_score.updated",
    "overdraft.risk_detected",
    "fuel_advance.triggered"
  ],
  "secret": "wh_secret_your_signing_key"
}

// Webhook Payload Example — bridge.distribution_issued
{
  "event": "bridge.distribution_issued",
  "event_id": "EVT-20270115-00481",
  "timestamp": "2027-01-15T06:00:00Z",
  "data": {
    "investment_id": "BRG-20260901-INV-00341",
    "amount_usd": 76.25,
    "carbon_credits": 12,
    "carbon_credit_value_usd": 8.40
  }
}
